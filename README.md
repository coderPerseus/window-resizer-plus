# window resizer plus

浏览器窗口调整神器，只需一个 popup 就能快速设置窗口大小、比例与位置，适合开发、测试与设计场景。

## 主要功能

- 一键调整浏览器窗口大小与位置
- 支持自定义宽高与常用比例（如 16:9、4:3 等）
- 保存常用预设，快速切换不同窗口配置
- 轻量、无干扰，专注解决窗口尺寸管理

## 开发

建议使用 pnpm：

```bash
pnpm install
pnpm dev
```

## 常用命令

```bash
# 开发
pnpm dev
pnpm dev:firefox

# 构建
pnpm build
pnpm build:firefox

# 打包
pnpm zip
pnpm zip:edge
pnpm zip:firefox

# 质量检查
pnpm lint
pnpm format
pnpm check
pnpm compile
```
